webpack 安装与简单使用 淘宝npm镜像：npm install -g cnpm --registry=https://registry.npm.taobao.org 1.全局安装 cnpm install webpack -g cnpm install webpack-cli -g (脚手架) => cnpm install webpack webpack-cli -g 2.初始化，生成package.json文件（进入项目所在目录） npm init / npm init -y（使用默认配置） 3.局部安装 cnpm install webpack -S cnpm install webpack-cli -S => cnpm install webpack webpack-cli -S 4.打包 1)webpack 默认entry: src/index.js 默认output: dist/main.js 2)打包模式选择 mode 压缩代码： webpack --mode production（默认） 不压缩代码: webpack --mode development
